Tutorial Dasar Termux

> Untuk pemula yang mau ngoding dari HP Android

1. Install Package Dasar

Pertama update dulu Termux nya:

pkg update && pkg upgrade

Install Python dan Git:

pkg install python git

2. Akses Storage HP

Biar bisa buka file dari folder Download:

termux-setup-storage

3. Buat Folder Project

mkdir project-gue
cd project-gue
nano index.html

4. Jalankan Server Lokal

Test web lu langsung di HP:

python -m http.server 8080

> Buka browser, ketik: localhost:8080
