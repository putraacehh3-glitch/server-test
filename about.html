Tentang BENI DEV

BENI DEV adalah website pertama gue yang dibikin 100% pakai HP + Termux. Gak pake laptop, gak pake PC mahal. Cuma modal HP Android dan niat belajar.

Tujuan Website Ini:

1. Nunjukin kalau bikin website itu bisa dimulai dari HP aja.
2. Bagi-bagi tutorial step by step buat pemula yang mau belajar coding.
3. Ngumpulin orang-orang yang pengen belajar bareng dari nol.

Website ini masih tahap belajar dan bakal terus gue update. Kalau lu juga lagi belajar, gas bareng di sini. Ada saran atau mau nanya? Langsung ke halaman Kontak.

Tutorial Pertama

Bikin Heading & Paragraf di HTML

Di HTML, buat bikin judul gede pakai tag <h1> sampai <h6>. Buat nulis teks biasa pakai tag <p>.

Contoh Kodenya:

<h1>Ini Judul Besar</h1>
<p>Ini paragraf biasa. Gue bisa nulis apa aja di sini.</p>

Hasilnya di Browser:

Ini Judul Besar

Ini paragraf biasa. Gue bisa nulis apa aja di sini.

Gampang kan? Coba lu buka Termux, bikin file baru `coba.html`, terus paste kode di atas. Jalankan pakai `php -S localhost:8080` terus buka di browser.

Tutorial Kedua

Bikin Tombol & Link di HTML

Buat bikin link pakai tag <a>. Buat bikin tombol yang keliatan keren, kita bikin link tapi dikasih style CSS biar kayak tombol.

Contoh Kodenya:

<a href="https://google.com">Ini Link Biasa</a>

<a href="kontak.html" class="tombol">Ini Tombol</a>

<style>
.tombol {
  background:#ff0000;
  color:#fff;
  padding:10px 20px;
  border-radius:5px;
  text-decoration:none;
  display:inline-block;
}
</style>

Hasilnya di Browser:

Ini Link Biasa

Ini Tombol

Tag href itu buat ngatur mau ke mana linknya. Kalau mau link ke file lain di folder yang sama, tinggal tulis nama filenya aja kayak kontak.html.
